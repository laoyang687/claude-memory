## 写作风格特征

### 语言特征

| 特征 | 数据 | 说明 |
|-----|------|------|
| 直接称呼 | 每千字41次 | 高频使用"you/your" |
| 对话感强 | - | 像朋友一对一交谈 |
| 句子节奏 | 40/40/20 | 短/中/长句混合 |
| 平均句长 | 14-15词 | 最佳可读性 |
| 对比转折 | 44-89次/视频 | 频繁使用"but" |

### 心理技巧

- **社会对比**："大多数人做X，但你应该Y"
- **认知冲突**：建立"常见观点 vs 真相"对比
- **知识缺口**：激发好奇心和学习欲望
- **紧迫行动**：强调"今天就可以开始"

## 模板快速参考

### 开场白模板

**社会对比型（推荐）：**
```
"[大多数人/社会] [做/认为] X,
但 [实际上/事实上] [真相] 是 Y...

这之所以重要，是因为..."
```

**条件假设型：**
```
"如果你 [痛点],
那么我有个 [好消息/解决方案] 给你...

因为..."
```

**个人故事型：**
```
"当我 [情境], 我 [问题]...
然后我发现了 [洞察]..."
```

## 核心原则（10条）

1. ✅ **始终用"你"** - 一对一对话
2. ✅ **多用"但是"** - 认知冲突
3. ✅ **"大多数人"** - 社会对比
4. ✅ **句子简短** - 14-15词
5. ✅ **混合长短** - 创造节奏
6. ✅ **具体数字** - 增强可信
7. ✅ **个人故事** - 建立信任
8. ✅ **提供框架** - 不只观点
9. ✅ **行动步骤** - 促使行动
10. ✅ **对话感** - 像朋友交谈
