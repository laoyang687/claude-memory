## 开场白模板

### 模板 1.1: 社会对比型 ⭐ 最有效（44%使用率）

**英文模板：**
```
"[Most people/Society/Everyone] [do/think/believe] X,
[but/actually/however] [the truth/reality/fact] is Y...

This matters because [implication/importance/significance]."
```

**中文模板：**
```
"[大多数人/社会/人们] [做/认为/相信] X,
[但实际上/然而/事实上] [真相/现实/事实] 是 Y...

这之所以重要，是因为 [含义/重要性/意义]。"
```

**示例 1（个人品牌）：**
```
"Most people think they need to choose a single career path,
but actually the most successful people combine multiple skills...

This matters because AI makes single skills commoditized."
```

**示例 2（内容创作）：**
```
"大多数人认为每天发内容才能成功，
但实际上质量远比数量重要...

这之所以重要，是因为低质量内容会损害你的品牌。"
```

**使用场景：**
- ✅ 颠覆常见认知
- ✅ 建立差异化定位
- ✅ 吸引寻求突破的受众

Because we're living in the golden age of multipotentialites."

**示例 2（职业转型）：**

**使用场景：**

## 内容结构模板

### 模板 2.1: 标准视频结构（8分钟）

**英文模板：**
```
[0:00-0:30] THE HOOK
  ├─ Counterintuitive statement
  ├─ Surprising statistic
  └─ Personal story

[0:30-2:00] THE PROBLEM
  ├─ Why most people fail
  ├─ Systemic barriers
  └─ Empathetic validation

[2:00-5:00] THE FRAMEWORK
  ├─ Core concept
  ├─ 3-5 key points
  └─ Specific examples

[5:00-7:00] THE STRATEGY
  ├─ Step one
  ├─ Step two
  └─ Step three

[7:00-8:00] THE ACTION
  ├─ What to do TODAY
  ├─ Long-term vision
  └─ Closing inspiration
```

**中文模板：**
```
[0:00-0:30] 钩子
  ├─ 反常识观点
  ├─ 惊人数据
  └─ 个人故事

[0:30-2:00] 问题
  ├─ 为什么大多数人失败
  ├─ 系统性障碍
  └─ 共情式验证

[2:00-5:00] 框架
  ├─ 核心概念
  ├─ 3-5个关键点
  └─ 具体案例

[5:00-7:00] 策略
  ├─ 第一步
  ├─ 第二步
  └─ 第三步

[7:00-8:00] 行动
  ├─ 今天可以做什么
  ├─ 长期愿景
  └─ 结语激励
```

**英文模板：**
```

真正的方法是 [框架]:
1. [第一步]
2. [第二步]
3. [第三步]

今天就可以 [立即行动]。"
```

```

The real solution is the Audience-First Framework:

2. Share your journey publicly
3. Build an audience while you build skills

You can start today by posting one insight from what you learned."
```
