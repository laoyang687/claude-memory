     1→# 错题分析器
     2→
     3→> 面向中小学生的智能错题管理系统，帮助学生高效收集、分析、复习错题。
     4→
     5→## 项目简介
     6→
     7→本产品通过**拍照录入 + AI 分析 + 科学复习**的方式，帮助中小学生建立个性化错题本，定位薄弱知识点，提升学习效率。
     8→
     9→### 核心功能
    10→
    11→- **智能收录**：拍照 OCR 识别，快速录入错题
    12→- **深度分析**：AI 分析错误原因，定位薄弱知识点
    13→- **科学复习**：基于艾宾浩斯遗忘曲线的智能复习提醒
    14→- **数据驱动**：可视化学习报告，追踪进步轨迹
    15→
    16→### 使用场景
    17→
    18→| 用户 | 使用场景 | 访问方式 |
    19→|------|----------|----------|
    20→| 学生 | 拍照录题、查看错题、复习练习 | 微信小程序 |
    21→| 家长 | 查看孩子学习报告、督促复习 | 微信小程序 + 网页 |
    22→| 教师 | 查看班级错题统计、布置针对性练习 | 网页端 |
    23→

    65→## 技术栈
    66→
    67→### 后端
    68→- **框架**：NestJS / FastAPI
    69→- **数据库**：MySQL 8.0 + Redis
    70→- **ORM**：Prisma / TypeORM
    71→- **AI**：豆包 / 通义千问 API
    72→- **OCR**：腾讯云 OCR
    73→
    74→### 小程序
    75→- **框架**：原生小程序 / uni-app
    76→- **状态管理**：Pinia (uni-app)
    77→
    78→### 网页端
    79→- **框架**：Vue 3 + TypeScript
    80→- **UI 组件**：Element Plus / Ant Design Vue
    81→- **图表**：ECharts
    82→
    83→## 快速开始
    84→
    85→### 1. 数据库初始化
    86→
    87→```bash
    88→# 创建数据库并导入初始数据
    89→mysql -u root -p < docs/database/init.sql
    90→```
    91→
    92→### 2. 后端服务
    93→
    94→```bash
    95→cd backend
    96→npm install
    97→# 配置 .env 文件
    98→npm run start:dev
    99→```
   100→
   101→### 3. 小程序
   102→
   103→```bash
   104→cd miniprogram
   105→npm install
   106→# 使用微信开发者工具打开项目
   107→```
   108→
   109→### 4. 网页端
   110→
   111→```bash
   112→cd web
   113→npm install
   114→npm run dev
